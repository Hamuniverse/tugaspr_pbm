## Deskripsi Aplikasi
Aplikasi ini merupakan tugas praktikum mata kuliah Pemrograman Berbasis Mobile.
Aplikasi memiliki fitur manajemen produk yang terhubung dengan REST API, serta dilengkapi dengan autentikasi menggunakan Bearer Token.

## Fitur
- Login menggunakan NIM
- Menampilkan daftar produk (katalog)
- Menambahkan produk baru
- Menghapus produk (soft delete)
- Submit tugas

## Teknologi
- Flutter
- Dart
- REST API
- flutter_secure_storage
- HTTP Package
